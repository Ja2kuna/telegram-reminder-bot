# Telegram Bot "Напоминалка"

Простой Telegram-бот для создания напоминаний.  
Индивидуальный проект, 10 класс.

## Возможности
- Создание напоминаний с указанием времени (в минутах)
- Просмотр списка активных напоминаний
- Отметка о выполнении
- Удаление задач

## Команды
| Команда | Описание |
|--------|----------|
| `/start` | Приветствие и список команд |
| `/remind [текст] [минуты]` | Создать напоминание (пример: `/remind Сделать уроки 60`) |
| `/list` | Показать все напоминания |
| `/done [номер]` | Отметить задачу выполненной |
| `/del [номер]` | Удалить напоминание |
| `/help` | Помощь |

## Пример использования
1. Пишешь боту: `/remind Купить хлеб 30`
2. Бот отвечает: "Напоминание добавлено! ID: 1"
3. Через 30 минут бот присылает: "🔔 Напоминание: Купить хлеб"
